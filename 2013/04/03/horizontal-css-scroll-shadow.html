Horizontal CSS Scroll Shadow

I recently simplified the design of this blog. It’s based on this blog by Stephanie Briones, a design craftsman at 8th Light.

One simple change I implemented made code examples horizontally scrollable on mobile phones. I liked this a lot, but felt the wide code examples should invite the user to scroll sideways.

I’ve recently spotted an effect that hides overflowing content beneath a shadow to give the visual cue that there is more information to be seen. I did some searching and found a great article by Lea Verou on how to accomplish this effect with CSS.

I adapted the code from the article to work with horizontally scrollable content and this is what I came up with.
