Resetting root MySQL password

I needed to setup a local development environment that uses MySQL. I had setup MySQL ages ago and forgot the root password. After some searching I found a helpful walkthrough. It’s just what I needed; I just changed some of the commands to match my set up.

I’ve added the commands I used below for posterity.

Basic steps:

  1. Kill server
  2. Start server without privilege system
  3. Connect to MySQL and reset password
  4. Kill the server without privilges
  5. Restart server normally

The following steps shutdown the server, restart it without the privilege system and connects to MySQL as root.

    $ killall mysqld
    $ mysqld_safe --skip-grant-tables &
    $ mysql -u root

Now you’re inside the MySQL command line client.

    mysql> USE mysql;
    mysql> UPDATE user SET Password=PASSWORD('new-password') WHERE User='root';
    mysql> FLUSH PRIVILEGES;
    mysql> \c

You’re password is now reset. Restart your server and continue working.

    $ mysqld_safe &
