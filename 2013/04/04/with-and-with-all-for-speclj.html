with! and with-all! for Speclj

I submitted a pull request to the Speclj test framework that adds the with! (with bang) and with-all! spec components. These allow you to create a referencable symbol that are evaluated immediately, rather than lazily as their non-bang counterparts; with and with-all.

An example of using with! in Speclj.

    (describe "with!"
      (def counter (atom 0))
      (with! with-example (swap! counter inc))

      (it "has deref'ed with-example and incremented the counter"
        (should= 1 @counter))

      (it "increments counter again after resetting with-example"
        (should= 2 @counter))

      (it "can be accessed as usual"
        (should= 3 @with-example)))

    (run-specs)

An example of using with-all!

    (describe "with-all!"
      (def counter (atom 0))
      (with-all! with-all-example (swap! counter inc))
      
      (context "example set #1"
        (it "has deref'ed with-all-example and incremented the counter"
          (should= 1 @counter))

        (it "only resets with-all-example once per context"
          (should= 1 @counter)))

      (context "example set #2"
        (it "still doesn't reset or deref with-all-example"
          (should= 1 @counter))))

    (run-specs)
