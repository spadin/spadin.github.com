Colorize you cats

Working constantly in the command-line, I’ve found printing code using cat slows me down because I’m used to the colorized output of vim.

I recently install Pygments and thought this would be a nice use for the tool. A Google search later, I found a command to colorize a file on the command-line.

    $ pygmentize -f terminal -g

I wasn’t ready to commit to learning such a long command so I created a simple alias:

    alias kat='pygmentize -f terminal -g'

Now when I need to take a peek at a file, I simply use:

    $ kat [filename]

If you’d like to give this a try, just add the alias above to your ~/.bashrc, ~/.zshrc or appropriate file. You’ll need to install Pygments. On my Mac, I installed Pygments by using the easy_install tool:

    $ easy_install Pygments
