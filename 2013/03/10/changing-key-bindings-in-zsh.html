Changing Key Bindings in Zsh

Key bindings in zsh with bindkey

I’ve recently had a newfound interest in trying to learn more about how exactly the tools I use on a day-to-day basis work. One of these tools is my terminal; in particular, I use the Zshell, or zsh.

One way that I use of traversing through my prompt is Option-Left arrow and Option-Right arrow to jump from word to word on the line. On occassion, I find the need to jump to the beginning or end of the line and end up either jumping word for word or just pressing and holding the arrow key. While I supposed either of these are fine, I thought there should be a quicker way to jump to the beginning or end of a line.

I began looking into key bindings, and this is what I found. These tips work well for me, using zsh and OS X, but as they say, your mileage may vary.

Finding a key sequence

In order to bind a key to a command you need two things, a key and a command. A way to find the key, also known as the escape sequence, is by following a simple series of steps.

Typing Ctrl-v on your prompt reads the next character literally and prints it. You can try it by typing:

    $ <ctrl-v><ctrl-vleft-arrow>


Note: actually enter Ctrl-v followed by the left arrow key.

Which should print:

    $ ^[[D


You’re output may be different depending on yourour operating system.

Regardless of the actual ouput, you can now bind that output to a certain command, so for example:

    $ bindkey '<ctrl-v><ctrl+left-arrow>' beginning-of-line
     $ bindkey '<ctrl-v><ctrl+right-arrow>' end-of-line


These will bind Ctrl+Left-arrow to move the cursor to the beginning of the line and Ctrl+Right-arrow to move the cursor to the end of the line.

To make these changes permanent, you would add these to lines to your .zshrc file.

    bindkey '^[[1;5D' beginning-of-line
     bindkey '^[[1;5C' end-of-line


I believe that Ctrl-Left arrow/Right arrow may be used in Linux distros to move forward/backward one word, so you may want to bind these two commands to some other key sequence.

Check the resources below for more detailed information and a list of possible commands you can use for binding.

tmux protip

If you notice these key bindings work as expected in Zsh, but stop working when using tmux, you may have to add this to your ~/.tmux.conf file.

    set-window-option -g xterm-keys on
